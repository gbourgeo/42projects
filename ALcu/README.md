# ALCU
*Awesome puissance 4 in terminal*
WoW.

# INSTALL
```sh
$> make && ./puissance4
Usage : ./puissance4 [line] [column]
```

# BEHAVIOUR
- 2 players MAY play this game
- Each turn put your coin in a column
- Align 4 of them to win the game !
